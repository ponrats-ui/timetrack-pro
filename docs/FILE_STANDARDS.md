# Future File Standards

This document records future file standard ideas only.

No package format is implemented yet.

## TimeTrack Package

Extension: `.ttp`

Purpose: a future TimeTrack Pro package format for employee work records,
payroll-ready summaries, import metadata, and HR handoff.

The package should help TimeTrack Pro users exchange structured employee data
without relying only on copy/paste or spreadsheets.

## Business OS Package

Extension: `.bos`

Purpose: a future Business OS package format for cross-Assistant data exchange.

The package should support workflows that involve multiple Assistants, such as
Employee Assistant, Finance Assistant, Tax Assistant, and Founder Dashboard.

## Open Format Rule

Future package formats must not replace Excel, CSV, PDF, JSON, or email.

They should add structure for customers who need it while keeping familiar
formats available.
